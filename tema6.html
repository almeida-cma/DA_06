📖 O que são Vetores e Matrizes?

Vetores e Matrizes são estruturas que permitem armazenar múltiplos valores do mesmo tipo em uma única variável.

🎯 ANALOGIA PARA ENTENDER:
  • 📊 Vetor: Como uma fila de caixas numeradas. Cada caixa guarda um valor. Você acessa pelo número da caixa (índice).
  • 📋 Matriz: Como uma tabela com linhas e colunas. Você acessa um valor pela linha e coluna.

🎯 Para que servem?

  • 📊 Armazenar listas de dados (notas de alunos, temperaturas, etc.)
  • 📋 Organizar dados em tabelas (planilhas, jogos, imagens)
  • 🔄 Facilitar repetições (percorrer todos os elementos com loops)
  • 📈 Resolver problemas matemáticos e estatísticos

✅ Vantagens

  • ⚡ Acesso rápido: qualquer elemento pelo índice
  • 📦 Organização: dados agrupados de forma estruturada
  • 🔄 Reutilização: mesma lógica para qualquer tamanho de array

⚠️ Desvantagens

  • 📏 Tamanho fixo: não cresce automaticamente
  • 📊 Tipo único: só armazena um tipo de dado
⚠️ IMPORTANTE: Em Java, o primeiro elemento de um vetor está na posição 0 (ZERO)!

📊 Vetores (Arrays Unidimensionais)

Um vetor é uma coleção de elementos do mesmo tipo, armazenados em posições sequenciais (índices).

📝 Como declarar um vetor

// Três formas de declarar um vetor
int[] numeros;        // Forma 1 (recomendada)
int []numeros;        // Forma 2
int numeros[];        // Forma 3

🏗️ Como instanciar (criar) um vetor

// Criar um vetor com 5 posições
int[] numeros = new int[5];

// Ou declarar e instanciar separadamente
int[] idades;
idades = new int[10];

✏️ Como inicializar um vetor

// Forma 1: posição por posição
int[] numeros = new int[5];
numeros[0] = 10;
numeros[1] = 20;
numeros[2] = 30;
numeros[3] = 40;
numeros[4] = 50;

// Forma 2: direta (mais simples)
int[] valores = {10, 20, 30, 40, 50};

🔍 Como acessar elementos

int[] numeros = {10, 20, 30, 40, 50};

System.out.println(numeros[0]);  // 10
System.out.println(numeros[2]);  // 30
System.out.println(numeros[4]);  // 50

// ATENÇÃO: O índice começa em ZERO!

📏 Como saber o tamanho

int[] numeros = {10, 20, 30, 40, 50};
int tamanho = numeros.length;  // 5

System.out.println("O vetor tem " + tamanho + " elementos");

🔄 Percorrendo com for

int[] numeros = {10, 20, 30, 40, 50};

for (int i = 0; i < numeros.length; i++) {
    System.out.println("Posição " + i + ": " + numeros[i]);
}

🔄 Percorrendo com for-each (mais simples)

int[] numeros = {10, 20, 30, 40, 50};

for (int valor : numeros) {
    System.out.println(valor);
}
💡 DICA: Use for-each quando você só precisa ler os valores. Use for quando precisar modificar ou acessar pelo índice.

📋 Matrizes (Arrays Multidimensionais)

Uma matriz é um vetor de vetores. É como uma tabela com linhas e colunas.

📝 Como declarar uma matriz

// Matriz 2D (3 linhas e 3 colunas)
int[][] matriz = new int[3][3];

✏️ Como inicializar uma matriz

// Forma 1: posição por posição
int[][] matriz = new int[3][3];
matriz[0][0] = 1;
matriz[0][1] = 2;
matriz[0][2] = 3;
matriz[1][0] = 4;
matriz[1][1] = 5;
matriz[1][2] = 6;
matriz[2][0] = 7;
matriz[2][1] = 8;
matriz[2][2] = 9;

// Forma 2: direta (mais simples)
int[][] numeros = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

🔍 Como acessar elementos

int[][] numeros = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

System.out.println(numeros[0][0]);  // 1 (linha 0, coluna 0)
System.out.println(numeros[1][2]);  // 6 (linha 1, coluna 2)
System.out.println(numeros[2][1]);  // 8 (linha 2, coluna 1)

🔄 Percorrendo uma matriz

int[][] numeros = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Loop externo: linhas
for (int i = 0; i < numeros.length; i++) {
    // Loop interno: colunas
    for (int j = 0; j < numeros[i].length; j++) {
        System.out.print(numeros[i][j] + " ");
    }
    System.out.println();  // Pula linha
}

💻 Exemplos Práticos

📌 Exemplo 1: Média de notas com vetor

🎯 OBJETIVO: Aprender a armazenar notas em um vetor, percorrer seus elementos com for-each e calcular a média. 
package tema06;

public class ExemploMedia {
    public static void main(String[] args) {
        double[] notas = {7.5, 8.0, 6.5, 9.0, 8.5};
        double soma = 0;

        for (double nota : notas) {
            soma += nota;
        }

        double media = soma / notas.length;
        System.out.println("Média das notas: " + media);
    }
}

📌 Exemplo 2: Maior valor em um vetor

🎯 OBJETIVO: Aprender a encontrar o maior valor dentro de um vetor, comparando elementos com if. 
package tema06;

public class ExemploMaior {
    public static void main(String[] args) {
        int[] numeros = {10, 45, 23, 78, 12, 56};
        int maior = numeros[0];

        for (int i = 1; i < numeros.length; i++) {
            if (numeros[i] > maior) {
                maior = numeros[i];
            }
        }

        System.out.println("O maior valor é: " + maior);
    }
}

📌 Exemplo 3: Soma de uma matriz

🎯 OBJETIVO: Aprender a percorrer uma matriz com loops aninhados (for dentro de for) e somar todos os elementos. 
package tema06;

public class ExemploMatrizSoma {
    public static void main(String[] args) {
        int[][] numeros = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        int soma = 0;

        for (int i = 0; i < numeros.length; i++) {
            for (int j = 0; j < numeros[i].length; j++) {
                soma += numeros[i][j];
            }
        }

        System.out.println("Soma de todos os elementos: " + soma);
    }
}

📌 Exemplo 4: Passando vetor para método

🎯 OBJETIVO: Aprender a reutilizar código criando um método que recebe um vetor como parâmetro e processa seus dados. 
package tema06;

public class ExemploMetodo {
    public static void main(String[] args) {
        int[] valores = {10, 20, 30, 40, 50};
        int soma = somar(valores);
        System.out.println("Soma: " + soma);
    }

    public static int somar(int[] vetor) {
        int total = 0;
        for (int valor : vetor) {
            total += valor;
        }
        return total;
    }
}
