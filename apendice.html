📝 Declaração de Vetores em Java

Existem diferentes formas de declarar um vetor em Java. Todas são válidas!

🎯 OBJETIVO: Mostrar as 3 formas de declarar um vetor e qual é a mais recomendada.

✅ Formas de declarar:

// Forma 1: tipo[] nome (recomendada)
int[] numeros;

// Forma 2: tipo []nome
int []numeros;

// Forma 3: tipo nome[]
int numeros[];
💡 DICA: Use a Forma 1 (int[] numeros) porque é a mais comum e legível!

🏗️ Como instanciar (criar) com tamanho:

// Criar um vetor com 5 posições
int[] numeros = new int[5];

// Ou declarar e instanciar separadamente
int[] idades;
idades = new int[10];

✏️ Como inicializar diretamente:

// Forma 1: com valores diretos (mais comum)
int[] numeros = {10, 20, 30, 40, 50};

// Forma 2: com new + valores
int[] numeros = new int[]{10, 20, 30, 40, 50};

// Forma 3: posição por posição
int[] numeros = new int[5];
numeros[0] = 10;
numeros[1] = 20;
numeros[2] = 30;
numeros[3] = 40;
numeros[4] = 50;
✅ IMPORTANTE: A questão 9 do quiz perguntava sobre isso!
Ambas as formas são válidas: int[] v = new int[]{10,20,30,40,50}; e int[] v = {10,20,30,40,50};

📦 Vetores são Objetos em Java

Vetores em Java são OBJETOS, não tipos primitivos!

🎯 OBJETIVO: Explicar que vetores têm atributos (como length) e podem ser passados para métodos.
⚠️ QUESTÃO 13 DO QUIZ:
Muitos alunos erram achando que vetor é tipo primitivo. NÃO É! Vetor é um OBJETO!

🔬 Por que vetores são objetos?

  • 📦 Têm atributos: length (tamanho)
  • 📦 Podem ser passados para métodos: minhaFuncao(vetor)
  • 📦 São criados com new: new int[5]
  • 📦 Possuem métodos: clone(), toString()

📝 Comparando tipo primitivo vs objeto:

CaracterísticaTipo primitivo (int)Vetor (int[])
Criado comValor direto: int x = 5;new int[5]
Possui atributos?NãoSim (length)
Possui métodos?NãoSim (clone(), toString())
Passado para métodoCópia do valorReferência ao objeto

💻 Exemplo prático:

package tema06;

public class ExemploVetorObjeto {
    public static void main(String[] args) {
        // Vetor é um objeto! Veja:
        int[] numeros = {10, 20, 30, 40, 50};

        // Atributo length (objeto)
        System.out.println("Tamanho do vetor: " + numeros.length);

        // Verificando se é um objeto: instanceof
        if (numeros instanceof Object) {
            System.out.println("Vetor é um objeto!");
        }

        // Passando vetor para um método
        int soma = somarVetor(numeros);
        System.out.println("Soma: " + soma);
    }

    // Método que recebe um vetor (objeto) como parâmetro
    public static int somarVetor(int[] vetor) {
        int total = 0;
        for (int valor : vetor) {
            total += valor;
        }
        return total;
    }
}
📤 Saída:
Tamanho do vetor: 5
Vetor é um objeto!
Soma: 150

📋 Imprimindo Vetores com Arrays.toString()

Para imprimir um vetor de forma fácil e legível, use Arrays.toString().

🎯 OBJETIVO: Ensinar a importar e usar Arrays.toString() para imprimir vetores de forma elegante.
⚠️ QUESTÃO 16 DO QUIZ:
Para usar Arrays.toString(), você precisa importar:
import java.util.Arrays;

📝 Como usar:

// Importe no início do arquivo
import java.util.Arrays;

// Depois use:
int[] numeros = {10, 20, 30, 40, 50};
System.out.println(Arrays.toString(numeros));
// Saída: [10, 20, 30, 40, 50]

📊 Comparação: Sem vs Com Arrays.toString()

package tema06;

import java.util.Arrays;  // ← IMPORTANTE!

public class ExemploArraysToString {
    public static void main(String[] args) {
        int[] numeros = {10, 20, 30, 40, 50};

        // SEM Arrays.toString() → resultado feio
        System.out.println("Sem toString: " + numeros);
        // Saída: [I@15db9742 (endereço de memória)

        // COM Arrays.toString() → resultado bonito!
        System.out.println("Com toString: " + Arrays.toString(numeros));
        // Saída: [10, 20, 30, 40, 50]
    }
}
📤 Saída:
Sem toString: [I@15db9742
Com toString: [10, 20, 30, 40, 50]
💡 DICA: Use Arrays.toString() sempre que precisar imprimir um vetor para depuração ou visualização!

📏 O que é matriz.length?

Em uma matriz, length retorna o número de linhas, não o total de elementos!

🎯 OBJETIVO: Esclarecer que matriz.length dá o número de LINHAS e como acessar o número de COLUNAS.
⚠️ QUESTÃO 8 DO QUIZ:
Muitos alunos erram achando que matriz.length dá o total de elementos. NÃO! Dá o número de LINHAS!

📊 Entendendo:

CódigoSignificadoExemplo
matriz.lengthNúmero de linhasint[][] m = new int[3][4]; → 3
matriz[0].lengthNúmero de colunas da linha 0int[][] m = new int[3][4]; → 4
matriz[i].lengthNúmero de colunas da linha iint[][] m = new int[3][4]; → 4

💻 Exemplo prático:

package tema06;

public class ExemploLengthMatriz {
    public static void main(String[] args) {
        int[][] matriz = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // matriz.length → número de LINHAS
        System.out.println("Número de linhas: " + matriz.length);       // 3

        // matriz[0].length → número de COLUNAS da linha 0
        System.out.println("Número de colunas: " + matriz[0].length);   // 3

        // Total de elementos = linhas × colunas
        int total = matriz.length * matriz[0].length;
        System.out.println("Total de elementos: " + total);             // 9
    }
}
📤 Saída:
Número de linhas: 3
Número de colunas: 3
Total de elementos: 9

⚠️ O que acontece ao acessar um índice inválido?

Ao tentar acessar uma posição que não existe no vetor, o Java lança uma exceção chamada ArrayIndexOutOfBoundsException.

🎯 OBJETIVO: Mostrar o que acontece ao acessar um índice fora do limite e como evitar esse erro.
⚠️ QUESTÃO 19 DO QUIZ:
Acessar numeros[5] em um vetor de tamanho 5 NÃO retorna null!
Gera um ERRO (ArrayIndexOutOfBoundsException)!

📊 Índices válidos:

int[] numeros = new int[5];  // tamanho = 5

// Índices válidos: 0, 1, 2, 3, 4
// Índices INVÁLIDOS: -1, 5, 6, ...

💻 Exemplo de erro:

package tema06;

public class ExemploErroIndice {
    public static void main(String[] args) {
        int[] numeros = {10, 20, 30, 40, 50};  // tamanho = 5

        System.out.println("Índice 0: " + numeros[0]);  // 10
        System.out.println("Índice 4: " + numeros[4]);  // 50

        // TENTANDO ACESSAR ÍNDICE 5 (NÃO EXISTE!)
        System.out.println("Índice 5: " + numeros[5]);  // ❌ ERRO!

        // O código abaixo NÃO executa!
        System.out.println("Este print não será exibido!");
    }
}
📤 Saída:
Índice 0: 10
Índice 4: 50
Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: Index 5 out of bounds for length 5
⚠️ OBSERVAÇÃO: O programa para de executar quando encontra esse erro. As linhas seguintes NÃO são executadas!

✅ Como evitar esse erro:

package tema06;

public class ExemploEvitarErro {
    public static void main(String[] args) {
        int[] numeros = {10, 20, 30, 40, 50};

        // Verifique se o índice é válido ANTES de acessar
        int indice = 5;

        if (indice >= 0 && indice < numeros.length) {
            System.out.println("Valor: " + numeros[indice]);
        } else {
            System.out.println("Índice inválido! O vetor tem " + numeros.length + " elementos.");
        }
    }
}
📤 Saída:
Índice inválido! O vetor tem 5 elementos.
