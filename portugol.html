📚 Dicionário - Vetores e Matrizes

Consulte sempre que tiver dúvida sobre o significado de uma palavra:

array → Vetor (coleção de elementos)
length → Comprimento (tamanho do vetor)
index → Índice (posição do elemento)
matrix → Matriz (tabela de dados)
row → Linha (na matriz)
column → Coluna (na matriz)
element → Elemento (cada valor do vetor/matriz)
for-each → Para cada (loop simplificado)
initialize → Inicializar (dar valores iniciais)
declare → Declarar (criar a variável)
instantiate → Instanciar (criar com new)
traverse → Percorrer (visitar todos os elementos)
